# AI Product Usage Analytics

An end-to-end analysis of user behavior, feature engagement, and churn patterns for an AI SaaS product.

## Tech Stack

- Python
- Pandas
- NumPy
- SQLite
- Matplotlib
- Jupyter Notebook

## Project Structure
```
AI_Product_Analytics/
│
├── AI_Product_Analysis_Project.ipynb
└── README.md
```

## How to Run

1. Clone the repository
2. Open `AI_Product_Analysis_Project.ipynb` in JupyterLab
3. Run all cells: Kernel → Restart & Run All

## Key Findings

- 46.6% churn rate — high for a SaaS product
- Top churn reason: switching to a competitor, not price
- prompt_editor has the highest session duration but is most associated with churned users
- India has the highest churn rate among all countries (55.6%)
- September 2025 had a significant churn spike
